# ТЗ — Nuances CRM

## Контекст

Юридическая CRM для фирмы «Нюансы». 2–3 юриста сейчас, до 10 в перспективе.
Разрабатывается итерационно — по одной фиче.

---

## Что уже есть (готово)

- [x] Авторизация (JWT)
- [x] Клиенты (физ. и юр. лица)
- [x] Дела (кейсы)
- [x] Документы + хранилище (MinIO/S3)
- [x] Задачи + календарь событий
- [x] Биллинг (учёт времени + счета + PDF)
- [x] Чат (WebSocket) — бэкенд готов, UI не допилен
- [x] Аудит действий
- [x] Роли: admin / lawyer
- [x] Docker (Postgres, Redis, MinIO)

## Архитектура

Модульный монолит + отдельный сервис для Telegram-бота.
Микросервисы избыточны для 2–10 пользователей.

```
┌─────────────────────────────────────────┐
│           Nginx (reverse proxy)          │
└────────────┬────────────────────────────┘
             │
    ┌────────┴─────────┐
    ▼                  ▼
┌──────────┐    ┌──────────────┐
│ Frontend │    │ Django API   │
│ Vue 3    │    │ (монолит)    │
└──────────┘    └──────┬───────┘
                       │
        ┌──────────────┼──────────────┐
        ▼              ▼              ▼
  ┌──────────┐  ┌──────────┐  ┌──────────┐
  │PostgreSQL│  │  Redis   │  │  MinIO   │
  │          │  │(channels │  │  (S3)    │
  └──────────┘  │ + cache) │  └──────────┘
                └──────────┘
                       │
                       ▼
              ┌─────────────────┐
              │  Telegram Bot   │
              │  (aiogram 3)    │
              └─────────────────┘
```

---

## Стек

| Слой | Технология |
|---|---|
| Frontend | Vue 3 + Vuetify 3 + Pinia |
| Backend | Django + DRF + Channels |
| База данных | PostgreSQL 16 |
| Кэш / очереди | Redis |
| Файлы | MinIO (S3-совместимый) |
| Real-time | Django Channels (WebSocket) |
| Telegram | aiogram 3 (отдельный контейнер) |
| Сервер | Daphne (ASGI) + Nginx |
| Деплой | Docker Compose |
